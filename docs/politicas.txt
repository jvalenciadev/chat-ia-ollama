POLÍTICAS DE JP-VALENCIA

1. Pagos
   - Se requiere un adelanto mínimo del 30% para iniciar cualquier proyecto.
   - El proyecto debe estar cancelado al 100% para su entrega final.

2. Tiempos de Entrega
   - Los tiempos varían según la complejidad y carga de trabajo.
   - Las fechas estimadas siempre se coordinan antes del inicio del proyecto.

3. Reembolsos
   - No se realizan reembolsos del adelanto si el cliente cancela el proyecto.
   - Si la cancelación es por parte de JP-VALENCIA, se devuelve el 100%.

4. Cambios y Ajustes
   - Se permiten pequeñas modificaciones dentro del alcance acordado.
   - Cambios grandes generan costos adicionales.

5. Soporte
   - Los proyectos incluyen 7 días de soporte gratuito.
   - Soporte extendido tiene costo adicional.

6. Confidencialidad
   - Toda la información del cliente se maneja de forma privada y segura.
